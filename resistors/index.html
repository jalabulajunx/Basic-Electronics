Color Code

Resistors use colored bands to show their value. Each color represents a number!

  • Brown = 1
  • Red = 2
  • Orange = 3
  • Yellow = 4

Series vs Parallel

In series circuits, resistors add up: Rtotal = R1 + R2

In parallel circuits: 1/Rtotal = 1/R1 + 1/R2

Fun Fact

Did you know? The first resistors were made from carbon and were used in early telephones!
